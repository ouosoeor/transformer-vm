## 🧭 What transformer-vm does
transformer-vm lets you compile programs into transformer weights. In plain terms, it turns a program into a model-like format that can run through a transformer-based engine.

It also includes a 2D convex-hull KV cache. That helps the app manage past data during inference with a fast lookup path. The goal is to keep response time low as the input grows.

This project is built for users who want to try a different kind of runtime on a normal Windows PC.

## ✨ Main features
- Compile programs into transformer weights
- Run transformer-based program logic on Windows
- Use a 2D convex-hull KV cache for faster inference
- Support for wasm-based workloads
- Built around a turing-mac style workflow
- Designed for local use on a desktop system
